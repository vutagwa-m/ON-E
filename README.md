#Project ON:E (Opportunity & Network for Equity)

> *Turning workplace barriers into bridges.*

**Project ON:E** is an AI-driven "Accommodation Passport" designed to bridge the gap between People with Disabilities (PwDs) and inclusive employers in Kenya and across Africa. 

### 🚀 The Problem
In Kenya, while the 5% disability hiring mandate exists, private sector representation remains below 1%. This is fueled by the "Disclosure-Accommodation Gap"—candidates fear bias, and employers fear the unknown costs of accommodation.

### ✨ The Solution
Built for the **AI for Accessibility Challenge**, Project ON:E removes the friction of disability disclosure by:
* **The Access Passport:** A functional profile (not a medical diagnosis) that defines a user's workplace needs.
* **AI Accommodation Engine:** Powered by Gemini, it parses Job Descriptions and provides a "Business Readiness Report" for employers, listing specific, low-cost tools and local African-context solutions.
* **Inclusion Analytics:** Data-driven insights to help organizations move from "compliance" to "true inclusion."

### 🛠 Tech Stack
* **Frontend:** Flutter (Accessibility-first UI/UX)
* **Backend:** FastAPI (Python)
* **Intelligence:** Google Gemini API (LLM for JD-to-Accommodation mapping)
* **Database:** Supabase (Auth & Real-time storage)

### 🌍 Impact
By de-risking the hiring process, Project ON:E aims to unlock the 7% of GDP currently lost due to the exclusion of PwDs from the global workforce.
